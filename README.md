# learning-dojo

A monorepo for hands-on Claude Code and software development learning projects.

## Projects

| Project | Description |
|---|---|
| [medium-mcp](./medium-mcp/) | MCP server that reads Medium premium articles via Playwright |
| [devto-mcp](./devto-mcp/) | MCP server that searches and reads Dev.to articles via the public REST API |

## Structure

Each project lives in its own subfolder with its own `pyproject.toml`, tests, and README.
